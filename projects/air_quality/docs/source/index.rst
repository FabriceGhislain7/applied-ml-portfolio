Air Quality Sensor Calibration
==============================

This documentation describes the development workflow, architecture, and Python API for the Air Quality Sensor Calibration project.

The project is a reproducible baseline workflow for noisy air-quality sensor data. It focuses on data cleaning, exploratory analysis, leakage-aware feature construction, regression modelling, and scientific limitations.
